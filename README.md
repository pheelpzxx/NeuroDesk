# 🧠 NeuroDesk - Sistema de Gestão de Produtividade

Sistema inteligente de produtividade e otimização de workflow para o setor farmacêutico/administrativo, utilizando IA para gestão de tarefas.

## 🚀 Funcionalidades

- ✅ Automação de relatórios
- ✅ Gestão de produtividade com priorização inteligente
- ✅ Interface intuitiva inspirada em padrões farmacêuticos
- ✅ Alertas em tempo real
- ✅ Registro diário de tarefas
- ✅ Dashboard interativo

## 📋 Requisitos

- Python 3.8+
- MySQL 8.0+
- pip
